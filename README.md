# End-to-End Data Cleaning Pipeline in SQL for Layoffs Dataset

This project focuses on cleaning and standardizing a real-world layoffs dataset using SQL.

Key steps include:
- Creating staging tables to preserve raw data
- Removing duplicate records using window functions
- Standardizing categorical fields (company, location, industry, country)
- Handling NULL and blank values using self-joins
- Converting date fields to proper DATE data types
- Removing rows and columns with insufficient data for analysis

Tools: MySQL
